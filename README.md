# Buildo Headshots

> AI-аватары для резюме, соцсетей и LinkedIn за 30 секунд

Один из 6 продуктов экосистемы **Buildo** — ИИ-агентов для бизнеса.

## Что это

Бизнес-фото на аву — это ai-аватары для резюме, соцсетей и linkedin за 30 секунд.

## Стек

- **Next.js 14** + React 18
- **TypeScript 5.7**
- **Tailwind CSS 3.4**
- **Brand color:** `#7C3AED`

## Локальная разработка

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Деплой

Платформа: **Layero** (auto-deploy из main).
